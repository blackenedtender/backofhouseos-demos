# Cookbook OS Public-Safe Demo

Cookbook OS is a future module for turning scanned cookbook pages into reviewed, searchable recipe records.

This demo uses sample data. Real systems may run locally/private.

## Status

Future concept / not deployed yet.

## What It Shows

- Future recipe scanning workflow.
- Review before recipe truth.
- Searchable recipe record concept.

## What Is Not Included

- Real recipe notes.
- Private household data.
- Scanned cookbook pages.
- Personal archive material.

## Run Locally

From this folder:

```powershell
python -m http.server 4178
```

Then open:

```text
http://localhost:4178
```
