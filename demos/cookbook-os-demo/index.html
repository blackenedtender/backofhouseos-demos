Future BackOfHouseOS module

Cookbook OS

Cookbook OS is a future module for turning scanned cookbook pages into reviewed, searchable recipe records.

Demo status

This demo uses sample data. Real systems may run locally/private.

No real recipe notes, household data, scanned pages, or private archive material are included.

Future workflow

  1. Scan cookbook page
  2. Extract recipe draft
  3. Review ingredients and steps
  4. Approve searchable recipe record
  5. Recall for meal planning
